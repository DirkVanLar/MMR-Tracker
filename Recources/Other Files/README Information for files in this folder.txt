The Categories.txt file will determine how the tracker sorts location areas. 
Order the location areas in the file the same way you would like them ordered in the tracker. 
Including unused location areas is fine.

Name your Logic files like this to have the trakcer load them as presets:
For Casual Logic: REQ_CASUAL.txt
For Glitched Logic: REQ_GLITCH.txt
For WWR Logic: WWR Logic.txt
For OOTR Logic: OOT Logic.txt
